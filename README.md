Introduction
------------

Boiler plate code for single app web application using Twitter Bootstrap.

Modules used on backend:

* express
* supervisor

Libraries used on frontend:

* underscore
* modernizr
* bootstrap
* jQuery
* require
* text
* backbone


Install
-------

	git clone https://github.com/finik/boilerplate.git myproject
	cd myproject
	npm install

Run
---

	cd myproject
	npm start
